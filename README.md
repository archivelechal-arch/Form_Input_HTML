Form ini dibuat menggunakan elemen-elemen HTML seperti `<input>`, `<label>`, `<fieldset>`, serta tombol (Submit) dan (Reset).

Struktur Formulir
Formulir terdiri dari beberapa bagian utama:

Data Pribadi
| Kolom         | Keterangan |

| Nama Lengkap  | Masukkan nama lengkap sesuai identitas |
| Alamat Email  | Gunakan alamat email aktif (contoh: `nama@gmail.com`) |
| Kata Sandi    | Buat kata sandi rahasia |
| Usia          | Isi dengan angka umur |
| Tanggal Lahir | Pilih tanggal lahir dari kalender |


Jenis Kelamin
Pilih salah satu:
- Laki-laki
- Perempuan


Minat
Centang satu atau beberapa bidang yang kamu minati:
- AI  
- Web  
- IoT  
- Game  

Kontak & Portofolio
| Kolom           | Keterangan 
| Nomor Telepon   | Masukkan nomor HP aktif 
| Link Portofolio | Tautan ke website atau proyek pribadi 

Upload Foto Profil
Klik (Choose File) lalu pilih file foto dari perangkat kamu  
(File yang disarankan: `.jpg`, `.jpeg`, atau `.png`)

Tombol Aksi
| Tombol | Fungsi 
| Submit |Mengirim data ke halaman `submit.html` 
| Reset  |Menghapus seluruh input agar form kosong kembali

Tips Pengisian
- Pastikan semua kolom diisi dengan benar sebelum menekan (Submit).  
- Gunakan (Reset) jika ingin mengulang dari awal.  
- Aktifkan `autocomplete="on"` agar pengisian jadi lebih cepat.
